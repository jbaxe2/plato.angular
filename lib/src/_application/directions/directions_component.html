Step 1. User Information:
Fill in your Plato username and password, then click the button to authenticate. Depending on how you arrived to this page, this authentication step may be performed for you; on such cases, simply start at Step 2.

Step 2. Courses Selection:
After authenticating, click on the Step 2. Courses Selection tab. Select a department, term, and course. Doing so will display the sections offered. Select one or more sections, and click the Add Selected Sections to the Request button. These sections will appear to the right, in the Requested Sections box. Repeat Step 2 as needed.

Step 3. Requested Sections:
The Requested Sections box will display the list of courses you are requesting to have created in Plato. At the bottom of each requested course will be icon button controls to copy previous content (which now includes archived course content), add the course to a cross-listing set, and remove that particular course from the request. The Icon Button Legend at the top of the Requested Courses box identifies which icon button corresponds to which control.

Step 4. Review & Submit Course Request:
Be sure to add any wanted previous content or specify cross-listing sets before doing this step! (See Step 3)

Click the Step 4. Review & Submit Course Request button (this button will display below the box for Steps 1 & 2 once one or more courses have been added to the course request) to review what will be submitted for your course request. If the request looks accurate, click on the Submit Course Request button.
